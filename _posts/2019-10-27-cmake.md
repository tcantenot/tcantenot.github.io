## CMake, Clang and Visual Studio

Install the Clang toolset following the instructions at
<https://github.com/tcantenot/llvm-utils>

In CMake-Gui, input the LLVM toolset (e.g *LLVM_v142*) to use under *Optional toolset to use (argument -T)*
